# ADR-001: Registro de decisões de arquitetura

- Status: Aceito
- Data: 2026-07-19

## Contexto

O projeto Cripto Bot é um protótipo experimental de automação de trading e envolve decisões técnicas que afetam manutenção, evolução e entendimento do sistema. Como o projeto ainda está em fase inicial, é importante registrar escolhas importantes para evitar ambiguidades futuras.

## Decisão

Registrar as decisões arquiteturais em arquivos Markdown no diretório docs/adr, com numeração sequencial e estrutura padronizada.

Cada ADR deve conter:

- Status e data;
- contexto da decisão;
- opção escolhida;
- consequências esperadas;
- status da decisão.

## Consequências

Isso melhora a compreensão do projeto para novos colaboradores e facilita futuras mudanças sem perder o histórico das decisões adotadas.
